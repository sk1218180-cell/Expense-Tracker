# 💰 Expense Tracker

A full-stack Expense Tracker application built using the MERN stack. It enables users to securely manage their income and expenses, track financial transactions, and monitor their financial activities through a clean and responsive interface.

---

## ✨ Features

- User Authentication (Register & Login)
- Add Income
- Add Expenses
- Edit Transactions
- Delete Transactions
- Dashboard with Financial Summary
- Recent Transactions
- Export Income and Expense Data to Excel
- Responsive User Interface

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- React Router DOM
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- CORS
- dotenv

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/sk1218180-cell/Expense-Tracker
```

### Install Dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd ../frontend
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the `backend` folder.

```env
PORT=4000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm run dev
```
